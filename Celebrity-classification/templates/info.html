    Аннотация

    На создание этого проекта вдохновила miracl6 (YouTube).

    Данный веб-сервис позволяет определить на кого из кинозвезд по лицу похож человек. После загрузки фотографии предобученная модель выводит топ 5 самых лучших вариантов с нормализованными значениями точности по полученным актерам\актрисам. Для реализации проекта был использован язык программирования Python.

    Этапы реализации

    • Сбор датасета
    • Получение векторных представлений изображений
    • Обучение модели
    • Создание веб-приложения
    • Развёртывание на хостинге

    Сбор датасета

    Для составления датасета была использована библиотека Selenium, при помощи которой через поисковик Google осуществлялся автоматический поиск и скачивание картинок. В итоге было собрано по 20 картинок для каждой из 135 знаменитости. Однако при таком сборе не все файлы оказались валидными: некоторые не открывались, на некоторых было несколько лиц.

    Получение векторных представлений

    Далее каждая картинка была представлена в виде 128-размерного вектора чисел с помощью библиотеки face_recognition. Также с помощью нее отсеивались изображения с несколькими лицами. Таким образом теперь можно было обучить модель на этих векторах.

    Обучение модели

    На полученных данных был обучен catboost классификатор. На тестовых данных метрика F1-score = 0.87.

    Создание веб-приложения

    Серверная часть реализована на Flask. Для фронта был использован Bootstrap. Загруженные картинки обрабатываются онлайн без сохранения где бы то ни было.

    Развёртывание на хостинге

    В качестве хостинговой платформы была выбрана Heroku из-за бесплатного тарифа и простоты развертывания.
